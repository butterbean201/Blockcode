# Blockcode
The Shack Initiative landing page
Black Code is a full-stack geospatial archive and 3D visualization engine designed to audit and reconstruct land succession histories in the American South. By synthesizing "dry" data—Bureau of Land Management (BLM) patents, Public Land Survey System (PLSS) coordinates, and local hardware ledgers—this project transforms flat records into an immersive 3D environment.
The initiative serves as a digital counter-script to the historical "Black Codes" of Mississippi, using technology to verify ancestral land ownership and visualize the systemic architecture of disenfranchisement.
🛠 The Stack
Backend: The Logic Layer (Laravel / SQL)
Relational Mapping: Architecting a database that links disparate historical datasets (Census records, DNA matches, and Land Deeds) into a single source of truth.
Forensic Auditing: Custom scripts to parse 19th-century land transfers and identify "breaks" in the chain of title.
Frontend: The Interface (React / Tailwind)
Data Visualization: Interactive timelines that track family migration and land loss from 1865 to the present.
The Advocacy Dashboard: A responsive UI for descendants to access and contribute to their own genealogical data.
Experience: The Spatial Layer (A-Frame / Three.js)
3D Succession Maps: Translating GPS coordinates from historical patents into 3D environments.
The Virtual Shack: Using VR to provide a physical sense of "place" for families whose ancestral homes no longer stand.
